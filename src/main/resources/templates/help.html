Help - Projects Tracker

Help & Documentation

Creating Your First Project
  1. Click the "Add New" button in the navbar
  2. Select "New Project"
  3. Fill in the project details (title is required)
  4. Select appropriate tags
  5. Click "Create Project"

Tip: Use tags to organize your projects by technology, client, or category.

Project Status
  • Not Started - Project is planned but not begun
  • In Progress - Currently working on this project
  • Done - Project is completed
Viewing Projects

Toggle between Card View and List View using the buttons in the toolbar.

Editing & Deleting

Click on a project card to view details, then use the Edit or Delete buttons.

Search Bar

Use the search bar in the navbar to find projects by title.

Advanced Filters
  • Status: Filter by NOT_STARTED, IN_PROGRESS, or DONE
  • GitHub: Show only projects on GitHub or local only
  • Tags: Click tag badges to filter by technology
  • Dates: Filter by creation month or last worked month
Sorting

Sort projects by:

  • Last Worked (most recent first)
  • Created Date (newest first)
  • Title (A-Z)

Creating Tags
  1. Go to the Tags page
  2. Enter a tag name (e.g., "Spring Boot")
  3. Choose a color
  4. Click "Create Tag"
Best Practices
  • Use consistent naming (e.g., "React" not "react" or "ReactJS")
  • Choose distinct colors for easy recognition
  • Create tags for technologies, clients, or project types

Export Options
  • CSV: For spreadsheet programs (Excel, Google Sheets)
  • HTML Report: Styled report with statistics

Exports respect your current filters! Apply filters before exporting to get specific data.

Click the / button in the navbar to toggle between light and dark modes.

Your preference is saved automatically and will be remembered on your next visit.
