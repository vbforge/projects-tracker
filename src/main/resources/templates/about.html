About Projects Tracker

A modern, intuitive project management application built with Spring Boot and Thymeleaf.

Features

  • Create and manage projects
  • Tag-based organization
  • Advanced filtering and sorting
  • Export to CSV and HTML
  • Dark mode support
  • Responsive design

Technology Stack

Backend
  • Spring Boot 3
  • Spring Data JPA
  • Thymeleaf
  • MySQL / H2 Database
Frontend
  • Bootstrap 5
  • Bootstrap Icons
  • Custom CSS
  • JavaScript (ES6+)

v1.0.0 Released: February 2026
